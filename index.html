命令面板

本地 / CI 常用测试命令

基础安装

pip install -r scripts/requirements.txt
pip install -r requirements-factor-lab.txt

核心单元测试

python -m pytest research_core/factor_lab tests/ -v --tb=short

Submission 测试

python -m pytest submissions/momentum_20d/test_factor.py -v --tb=short

Alpha101 Demo

python -m research_core.factor_lab.cli run-alpha101-demo \
  --factors alpha1,alpha2 \
  --n-dates 160 --n-codes 8 --seed 7

Factor Set Demo

python -m research_core.factor_lab.cli run-factor-set-demo \
  --factor-set alpha158 \
  --n-dates 160 --n-codes 8 --seed 7

CI 校验脚本

python scripts/ci_detect_changes.py --base-ref origin/main
python scripts/ci_factor_validate.py \
  --changed-json /tmp/changed_factors.json \
  --output-dir /tmp/validation_output
